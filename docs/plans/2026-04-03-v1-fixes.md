## Fix 1: Bearer Token Resolution Middleware (C1)

The most critical gap. MCP tools need to authenticate the incoming ChatGPT bearer token and set up the PCO client context.

**Files:**
- Create: `src/pco_mcp/middleware.py`
- Modify: `src/pco_mcp/main.py` — wire middleware into the MCP app
- Modify: `src/pco_mcp/tools/_context.py` — no changes needed, already has `set_pco_client`
- Create: `tests/test_middleware.py`

**What the middleware does:**
1. Intercepts requests to `/mcp/` endpoints
2. Reads `Authorization: Bearer <token>` header
3. Hashes the token with SHA-256
4. Looks up `OAuthSession` by `chatgpt_access_token_hash`
5. Checks `expires_at` — rejects if expired
6. Loads the associated `User`
7. Checks `pco_token_expires_at` — if expired, calls `refresh_pco_token` and updates the DB
8. Decrypts the PCO access token
9. Creates a `PCOClient` and calls `set_pco_client()`
10. On any failure, returns a plain-English error

**Implementation:**

```python
# src/pco_mcp/middleware.py
import hashlib
import logging
from datetime import UTC, datetime, timedelta

from fastapi import Request
from fastapi.responses import JSONResponse
from sqlalchemy import select
from sqlalchemy.ext.asyncio import async_sessionmaker

from pco_mcp.crypto import decrypt_token, encrypt_token
from pco_mcp.models import OAuthSession, User
from pco_mcp.oauth.pco_client import refresh_pco_token
from pco_mcp.pco.client import PCOClient
from pco_mcp.tools._context import set_pco_client

logger = logging.getLogger(__name__)


class BearerTokenMiddleware:
    """Resolves ChatGPT bearer tokens to authenticated PCO clients."""

    def __init__(
        self,
        app,
        session_factory: async_sessionmaker,
        pco_api_base: str,
        pco_client_id: str,
        pco_client_secret: str,
        token_encryption_key: str,
    ):
        self.app = app
        self.session_factory = session_factory
        self.pco_api_base = pco_api_base
        self.pco_client_id = pco_client_id
        self.pco_client_secret = pco_client_secret
        self.token_encryption_key = token_encryption_key

    async def __call__(self, scope, receive, send):
        if scope["type"] != "http":
            await self.app(scope, receive, send)
            return

        request = Request(scope, receive)
        auth_header = request.headers.get("authorization", "")

        if not auth_header.startswith("Bearer "):
            await self.app(scope, receive, send)
            return

        token = auth_header[7:]
        token_hash = hashlib.sha256(token.encode()).hexdigest()

        try:
            async with self.session_factory() as db:
                # Look up session
                result = await db.execute(
                    select(OAuthSession).where(
                        OAuthSession.chatgpt_access_token_hash == token_hash
                    )
                )
                session = result.scalar_one_or_none()
                if session is None:
                    logger.warning("Bearer token not found in OAuthSession")
                    response = JSONResponse(
                        {"error": "Invalid or expired session. Please reconnect."},
                        status_code=401,
                    )
                    await response(scope, receive, send)
                    return

                if session.expires_at < datetime.now(UTC):
                    logger.info("Bearer token expired for session %s", session.id)
                    response = JSONResponse(
                        {"error": "Your session has expired. Please reconnect."},
                        status_code=401,
                    )
                    await response(scope, receive, send)
                    return

                # Load user
                user_result = await db.execute(
                    select(User).where(User.id == session.user_id)
                )
                user = user_result.scalar_one_or_none()
                if user is None:
                    logger.error("OAuthSession %s references missing user %s", session.id, session.user_id)
                    response = JSONResponse(
                        {"error": "Account not found. Please reconnect."},
                        status_code=401,
                    )
                    await response(scope, receive, send)
                    return

                # Refresh PCO token if needed
                pco_access_token = decrypt_token(user.pco_access_token_enc, self.token_encryption_key)
                if user.pco_token_expires_at < datetime.now(UTC) + timedelta(minutes=5):
                    logger.info("Refreshing PCO token for user %s", user.id)
                    pco_refresh = decrypt_token(user.pco_refresh_token_enc, self.token_encryption_key)
                    new_tokens = await refresh_pco_token(
                        refresh_token=pco_refresh,
                        client_id=self.pco_client_id,
                        client_secret=self.pco_client_secret,
                    )
                    pco_access_token = new_tokens["access_token"]
                    user.pco_access_token_enc = encrypt_token(pco_access_token, self.token_encryption_key)
                    user.pco_refresh_token_enc = encrypt_token(new_tokens["refresh_token"], self.token_encryption_key)
                    user.pco_token_expires_at = datetime.now(UTC) + timedelta(seconds=new_tokens.get("expires_in", 7200))
                    user.last_used_at = datetime.now(UTC)
                    await db.commit()

                # Set context
                client = PCOClient(base_url=self.pco_api_base, access_token=pco_access_token)
                set_pco_client(client)

        except Exception:
            logger.exception("Error resolving bearer token")
            response = JSONResponse(
                {"error": "Authentication error. Please try reconnecting."},
                status_code=500,
            )
            await response(scope, receive, send)
            return

        await self.app(scope, receive, send)
```

Wire into `main.py` by wrapping the `mcp_app` before mounting:

```python
from pco_mcp.middleware import BearerTokenMiddleware

mcp_app_with_auth = BearerTokenMiddleware(
    app=mcp_app,
    session_factory=session_factory,
    pco_api_base=settings.pco_api_base,
    pco_client_id=settings.pco_client_id,
    pco_client_secret=settings.pco_client_secret,
    token_encryption_key=settings.token_encryption_key,
)
app.mount("/mcp", mcp_app_with_auth)
```

## Fix 2: OAuth Security Hardening (C2-C5)

Fix the 4 OAuth security criticals in `provider.py`.

**Files:**
- Modify: `src/pco_mcp/oauth/provider.py`
- Modify: `tests/test_oauth_provider.py`
- Modify: `tests/test_oauth_provider_extended.py`

**Changes:**

### C2: Refresh token validation
- Add `refresh_token_hash` column to `OAuthSession` model (modify `models.py`)
- Store hashed refresh token when issuing tokens
- Validate refresh token hash on refresh grant
- Rotate: invalidate old session, create new one
- Persist new access token as `OAuthSession`

### C3: PKCE enforcement
- In `/oauth/token`, when `code_challenge` is present in pending entry:
  - Require `code_verifier` form field
  - Compute `base64url(sha256(code_verifier))`
  - Compare with `secrets.compare_digest` against stored `code_challenge`
  - Reject with 400 if mismatch

### C4: Client credential validation
- In `/oauth/token`, look up `client_id` in `_registered_clients`
- Compare `client_secret` with `secrets.compare_digest`
- Reject with 401 on mismatch

### C5: Redirect URI validation
- In `/oauth/authorize`, look up `client_id` in `_registered_clients`
- Verify `redirect_uri` exactly matches one of registered URIs
- Reject with 400 if not found
